Full-stack engineer focused on backend architecture, cloud infrastructure, and maintainable systems.

## Engineering Focus

I care about building systems that are simple, observable, testable, and easy to change.

- Separating business rules from framework concerns
- Designing APIs with clear boundaries
- Keeping infrastructure reproducible with IaC
- Writing tests around critical domain behavior
- Choosing boring technology where reliability matters
